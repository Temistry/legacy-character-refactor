# 06. Public Release Checklist

Use this checklist before publishing the repository.

- [ ] Company name removed
- [ ] Game title removed
- [ ] Internal project name removed
- [ ] Real class names removed
- [ ] Real function names removed
- [ ] Real variable names removed
- [ ] Character names removed
- [ ] Skill names removed
- [ ] Named boss identifiers removed
- [ ] Server names removed
- [ ] Packet names removed
- [ ] Table names removed
- [ ] Region names removed
- [ ] Item names removed
- [ ] Internal file paths removed
- [ ] Internal comments removed
- [ ] Log messages anonymized
- [ ] Magic numbers replaced with toy values
- [ ] Real business logic replaced with simplified sample logic
- [ ] No copied source file from a company project
- [ ] No screenshot from an internal tool or editor
- [ ] No SVN history or commit metadata
- [ ] README clearly states this is a reconstructed educational sample
- [ ] Tests use only generic state and event names
- [ ] Documentation avoids implying ownership of a specific unreleased or internal system
