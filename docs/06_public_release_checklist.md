# 06. Public Release Checklist / 공개 전 체크리스트

## 사용 기준 / How To Use

GitHub에 공개하기 전에 이 목록을 확인한다.
체크리스트는 실제 회사 코드, 내부명, 도메인 지식이 남아 있는지 확인하는 용도다.

Use this checklist before publishing the repository.
The checklist is for finding remaining production code, internal names, or domain-specific knowledge.

## Checklist

- [ ] 회사명이 제거되어 있다. / Company name removed.
- [ ] 게임명이 제거되어 있다. / Game title removed.
- [ ] 내부 프로젝트명이 제거되어 있다. / Internal project name removed.
- [ ] 실제 class name이 제거되어 있다. / Real class names removed.
- [ ] 실제 function name이 제거되어 있다. / Real function names removed.
- [ ] 실제 variable name이 제거되어 있다. / Real variable names removed.
- [ ] character name이 제거되어 있다. / Character names removed.
- [ ] skill name이 제거되어 있다. / Skill names removed.
- [ ] named boss identifier가 제거되어 있다. / Named boss identifiers removed.
- [ ] server name이 제거되어 있다. / Server names removed.
- [ ] packet name이 제거되어 있다. / Packet names removed.
- [ ] table name이 제거되어 있다. / Table names removed.
- [ ] region name이 제거되어 있다. / Region names removed.
- [ ] item name이 제거되어 있다. / Item names removed.
- [ ] 내부 file path가 제거되어 있다. / Internal file paths removed.
- [ ] 내부 comment가 제거되어 있다. / Internal comments removed.
- [ ] log message가 익명화되어 있다. / Log messages anonymized.
- [ ] magic number가 toy value로 대체되어 있다. / Magic numbers replaced with toy values.
- [ ] 실제 business logic이 simplified sample logic으로 대체되어 있다. / Real business logic replaced with simplified sample logic.
- [ ] 회사 프로젝트의 source file을 복사하지 않았다. / No copied source file from a company project.
- [ ] 내부 tool 또는 editor screenshot을 포함하지 않는다. / No screenshot from an internal tool or editor.
- [ ] SVN history 또는 commit metadata를 포함하지 않는다. / No SVN history or commit metadata.
- [ ] README가 reconstructed educational sample임을 명시한다. / README clearly states this is a reconstructed educational sample.
- [ ] test는 generic state와 event name만 사용한다. / Tests use only generic state and event names.
- [ ] 문서는 특정 unreleased/internal system 소유를 암시하지 않는다. / Documentation avoids implying ownership of a specific unreleased or internal system.
